**Legend of states:** `IMPLEMENTED` · `PARTIALLY_IMPLEMENTED` · `CONTRACT_ONLY` · `DEFERRED` · `NOT_APPLICABLE`

Only what is protected by working code **and** a passing test is marked `IMPLEMENTED`.

Stage 1 rows cover identity, authentication, scoped ledger, integrity chain, projections, PostgreSQL append-only enforcement, measured health snapshots, and migration safety.
